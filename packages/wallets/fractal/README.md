# `@trezoa/wallet-adapter-fractal`

This package provides an adapter to enable Trezoa apps to connect to a Fractal Wallet.

For quick setup, please refer to the trezoa-xyz/wallet-adapter [README](https://github.com/trezoa-xyz/wallet-adapter#quick-setup-using-react-ui)

Integrating with [Fractal's Developer APIs and SDKs](https://developers.fractal.is) is the easiest way to build web3 games.
